## Running the Project

1. Navigate to the project folder:
   ```bash
   cd TheBestestCarRental/TheBestestCarRental
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. To run the website:
   ```bash
   npm run dev
   ```
